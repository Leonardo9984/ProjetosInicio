# Kami's designers empresa ficticia 

## Sobre
esse projeto foi apresentado com a ideia de um desafio, que consiste em apenas usar o Tailwind
e evitar o uso de CSS nesse site. O desafio surgiu por um professor na area Programação web, incentivando a
uma forma de aprender de uma forma mais profunda o Tailwind que é uma ferramenta muito util que agiliza 
muito a jeito de se estilizar um site que fique bonito e legível.
